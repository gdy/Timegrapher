This app needs Safari to access the microphone and analyze audio files.

iOS does not allow opening local files in Safari directly. To use this app:

1Ask the sender for a link instead of a file attachment
2Open the link in Safari — everything will work

Or use a desktop browser (Chrome, Edge, Firefox) where local files work fine.
